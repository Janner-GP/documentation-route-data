# Ruta de SQL

SQL es uno de los lenguajes mas importantes del mundo de datos y del desarrollo backend. Aprenderlo bien te permite consultar informacion, construir reportes, mantener bases de datos relacionales y resolver problemas reales de negocio.

## Que encontraras en esta seccion

Esta ruta esta organizada para ir de lo esencial a lo profesional:

- Fundamentos de SQL
- Consultas basicas y operaciones CRUD
- Funciones y agregaciones
- Joins
- Subconsultas
- Funciones avanzadas
- Window Functions
- Normalizacion
- Indices y performance
- Transacciones
- Seguridad y roles
- Triggers
- SQL en el mundo real

## Que es SQL

SQL significa `Structured Query Language`. Es el lenguaje estandar para consultar y manipular bases de datos relacionales como PostgreSQL, MySQL, SQL Server, Oracle y SQLite.

Con SQL normalmente hacemos tareas como:

- Consultar datos
- Insertar nuevos registros
- Actualizar informacion existente
- Eliminar registros
- Crear tablas, indices, vistas y permisos

## Por que aprender SQL

- Es base para analisis de datos, ingenieria de datos y backend
- Se usa en entrevistas tecnicas y pruebas para empleo
- Es clave para trabajar con BI, dashboards y reportes
- Ayuda a pensar mejor la estructura de la informacion

## Recomendacion de estudio

Avanza en orden. Primero entiende bien tablas, relaciones y consultas simples. Luego pasa a joins, subconsultas y agregaciones. Finalmente estudia performance, seguridad, transacciones y casos reales de trabajo.
